# Machine Learning Mastery - Industry Internship

This repository contains the practical projects and exercises developed during my **One-Week Industry Internship Program on "Machine Learning Mastery"** at **Elkanio Research Labs** (conducted from **July 08, 2024, to July 15, 2024**). 

The program focused on applying machine learning and deep learning techniques to solve real-world problems, ranging from exploratory data analysis to building Artificial Neural Networks.

---

## 📂 Repository Structure & Project Breakdown

The repository is organized into four distinct industry-focused exercises:

### 📁 Excercise 1: Retail Sales Analysis
* **File:** `Retail_Sales_Analysis.ipynb`
* **Focus:** Exploratory Data Analysis (EDA), data cleaning, handling datasets, and uncovering retail business trends using Python data libraries.

### 📁 Excercise 2: House Price Prediction
* **File:** `House_Price_Prediction.ipynb`
* **Focus:** Regression modeling to predict real estate pricing based on key housing features.

### 📁 Excercise 3: Customer Segmentation
* **File:** `CustomerSegmentation_And_Analysis.ipynb`
* **Focus:** Unsupervised machine learning and clustering techniques (like K-Means) to segment target consumers based on behavioral data.

### 📁 Excercise 4: Diabetes Prediction using ANN
* **File:** `Diabetes_Prediction_using_ANN.ipynb`
* **Focus:** Deep learning application utilizing Artificial Neural Networks (ANNs) for binary classification and predictive healthcare analytics.
